# 🚀 3. KEY PROJECT – AI VERIFICATION SYSTEM

## Problem

A pharmaceutical company received more than 13,000 monthly reports from 100+ super stockists across 9 divisions.

Challenges:

* Data came in multiple formats (PDF, scanned images, CSV, Excel, JPEG, PNG)
* No standardized structure
* Manual verification required 10–12 days
* Required a team of 10–12 people
* High risk of human error

---

## Solution

Designed and built an AI-powered automation system to process and verify stock reports.

Key Components:

* Multi-format data ingestion pipeline
* Data extraction from PDFs and scanned images
* Schema standardization and normalization
* Automated comparison engine

Technology Used:

* Gemini API for intelligent document parsing
* Python for data processing and pipeline development

---

## Key Features

* Automated extraction from unstructured documents
* Conversion into standardized structured format
* Data normalization across divisions
* Intelligent comparison with master data
* Detection of mismatches:

  * Product mismatch
  * Division mismatch
  * Missing entries

---

## Impact

* Reduced manual effort significantly
* Improved verification accuracy
* Reduced processing time drastically
* Enabled scalable automation for large datasets
